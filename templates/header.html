Instrucciones de uso

A continuación nombraremos las reglas principales para el uso de este proyecto:
1. Indicar explicitamente el producto de los terminos. Por ejemplo: (1/2)xy = (1/2)*x*y
2. La potencia de cualquier termino es con (**). Por ejemplo: x^3 = x**3
3. La raiz cuadrada de cualquier termino es con sqrt(x). Ejemplo: sqrt(21)
4. El valor de las funciones trigonometricos es evaluada en radianes. Ejemplo: sin(pi) = 0. Nota: Es necesario escribir sin(x) para calcular el seno.
5. Para calcular el logaritmo natural de un termino es necesario usar ln(x) y si se desea calcular el logaritmo en otra base es necesario usar log(x, base).
6. Para usar el valor de Pi solo es necesario escribir pi.
7. Para usar el valor de e (constante de Euler) es necesario escribir el exponente del mismo en exp(x).

Nota: Es necesario colocar correctamente los intervalos de integración, es decir, que el limite inferior sea menor que el limite superior. En cualquier caso, el sistema le avisara de algun error.

Importante: Las opciones que soliciten f(x,y) permitiran funciones que contienen x y y, mientras que una opcion que solicite f(x) solo aceptara funciones dependientes de x y el ingreso de otro tipo de función ocasionara un error.
